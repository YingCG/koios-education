# Koios

This is the beginning of the Koios Educational Website!

1. Clone down repo
2. cd koios
3. npm install
4. npm run knex migrate:latest
5. npm run knex seed:run
6. npm run dev

MVP: Homepage, Edu Resources (Astronomy), satellite data in a db
